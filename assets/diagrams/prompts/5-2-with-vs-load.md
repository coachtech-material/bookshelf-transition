## プロンプト本文

```
Eager Loading の with と load の使い分けを Before / After で対比する、立体感のある概念図を作成してください。

【内容】
- 画面を左右に分割する
- 左側（with）: 立体的なデータベース（円柱を重ねた形）から、人物が「タスク」と日本語で書かれた箱と、それに付く関連の小箱「カテゴリ」「タグ」（いずれも日本語ラベル）を「一緒に」1回で取り出している。上に「with これから取る」、下に「取得と同時に関連も読み込む」とラベル
- 右側（load）: 人物がすでに「タスク」と書かれた箱を手に持っている状態。そこへ後から関連の小箱「カテゴリ」「タグ」を「付け足して」いる。上に「load もう持っている」、下に「取得済みのモデルに後付け」とラベル

【スタイル】
- フラットなインフォグラフィック／ベクター風を基調に、アイソメトリック（斜め上視点）で立体感を出す（データベース・箱に厚みを持たせる）
- 影はオブジェクト真下の単色フラット影のみ。ぼかし影・グラデーションは使わない
- 背景は #FFFFFF の単色。テクスチャ・しみ・もや・かすれを描かない
- 左（with）は青（#3B82F6）、右（load）はオレンジ（#F97316）のアクセントカラー。クリーンな線画
- 箱や物のラベルはすべて日本語で書く（「タスク」「カテゴリ」「タグ」。TASK や TAG のような英語表記にしない）。「with」「load」はそのまま英語でよい
- タイトルや無関係な英単語など、指定外の文字は入れない
- 16:9 横長レイアウト
```
