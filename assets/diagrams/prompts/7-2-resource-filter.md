# 7-2-resource-filter

**モデル**: pro | **アスペクト比**: 16:9 | **解像度**: 2k

## プロンプト本文

```
API Resource が「モデルの列から見せる列だけを選んで JSON に整形する」様子を、立体感のある概念図で表してください。

【内容】
- 左に立体的なモデルのカードを置き「モデル Task」とラベル。中に列の一覧（「id」「title」「status」「created_at」「updated_at」「user_id」）
- 中央に立体的なフィルター／通訳の装置を置き「TaskResource」とラベル
- 左のモデルから中央の装置へ列が流れ込む。装置を通って右へ出るのは選ばれた列だけ（「id」「title」「status」）。内部用の列（「created_at」「updated_at」「user_id」）は装置でせき止められ、赤い × で隠される
- 右に立体的な JSON カードを置き「JSON レスポンス」とラベル。中身は「id」「title」「status」だけ

【スタイル】
- フラットなインフォグラフィック／ベクター風を基調に、アイソメトリック（斜め上視点）で立体感を出す（カードや装置に厚みを持たせる）
- 影はオブジェクト真下の単色フラット影のみ。ぼかし影・グラデーションは使わない
- 背景は #FFFFFF の単色。テクスチャ・しみ・もや・かすれを描かない
- モデルは青（#3B82F6）、装置と通過する列はオレンジ（#F97316）、隠される列はグレー＋赤い ×。クリーンな線画、ラベルは日本語（列名・JSON はそのまま）
- タイトルや無関係な英単語など、指定外の文字は入れない
- 16:9 横長レイアウト
```
