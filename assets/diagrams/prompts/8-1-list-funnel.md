```
一覧 API が大量のレコードを「検索→絞り込み→ページネーション」で1ページ分に絞り込む様子を、漏斗（じょうご）の比喩で表す立体感のある概念図を作成してください。

【内容】
- 左に大量のタスクのカードが山積みになっている（数えきれないほど）。上に「全タスク（大量）」とラベル
- 中央に大きな立体的な漏斗（じょうご）を置き、上から下へ3段の関門を通す。各段にラベル: 上から「検索（キーワード）」「絞り込み（条件）」「ページネーション（1ページ・上限100）」
- 漏斗の下の出口から、きれいにそろった少数のカード（20件ぶん）が1束だけ出てくる。下に「1ページ分だけ返す」とラベル

【スタイル】
- フラットなインフォグラフィック／ベクター風を基調に、アイソメトリック（斜め上視点）で立体感を出す（カードや漏斗に厚みを持たせる）
- 影はオブジェクト真下の単色フラット影のみ。ぼかし影・グラデーションは使わない
- 背景は #FFFFFF の単色。テクスチャ・しみ・もや・かすれを描かない
- 大量の山はグレー寄りの青、漏斗と出てくる1ページ分はオレンジのアクセント。クリーンな線画、ラベルは日本語
- タイトルや無関係な英単語など、指定外の文字は入れない
- 16:9 横長レイアウト
```
