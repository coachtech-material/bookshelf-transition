## プロンプト本文

```
API のエラー応答の Before / After を対比する立体感のある概念図を作成してください。利用者はプログラムです。

【内容】
- 画面を左右に分割する
- 左側（Before）: API が HTML のエラーページ（404 ページの紙）を返している。受け取ったプログラム（ロボットやスマホアプリのアイコン）が困惑し、赤い × と「JSON 解釈に失敗・クラッシュ」とラベル。上に「HTML を返す」
- 中央に、左から右へ向かう矢印
- 右側（After）: API が JSON のカード（波括弧と「error」「404」）を返している。受け取ったプログラムが緑のチェックで正しく処理。上に「JSON を返す」。右端に小さくステータスコードの帯（「200」「201」「204」「404」「422」）を添える

【スタイル】
- フラットなインフォグラフィック／ベクター風を基調に、アイソメトリック（斜め上視点）で立体感を出す（紙・カード・端末に厚みを持たせる）
- 影はオブジェクト真下の単色フラット影のみ。ぼかし影・グラデーションは使わない
- 背景は #FFFFFF の単色。テクスチャ・しみ・もや・かすれを描かない
- 左（Before）はグレー寄り＋赤、右（After）は青（#3B82F6）とオレンジ（#F97316）＋緑チェック。クリーンな線画、ラベルは日本語（JSON・HTML・error・数字はそのまま）
- タイトルや無関係な英単語など、指定外の文字は入れない
- 16:9 横長レイアウト
```
